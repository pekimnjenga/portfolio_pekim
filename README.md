A high-performance, responsive portfolio built with a **Cyber-Luxe** aesthetic


## Live Access
**View the live site here:** [My Portfolio](https://portfolio-pekim.onrender.com)



## Tech Stack & Architecture
* **Backend:** Python (Flask)
* **Frontend:** HTML5, CSS3 (Custom Glassmorphism), Bootstrap 5, JavaScript(ES6+)
* **Deployment:** Render


## Key Features
* **System Diagnostic UI:** A custom-built "HUD" style interface with interactive glass cards.
* **Interactive HUD Feedback:** Utilizes CSS3 keyframes and pseudo-elements to provide real-time visual "glowing" feedback on user interaction with project modules.
* **Custom Glass-Morphism Engine:** A high-energy UI built with advanced CSS3 backdrop-filters and deep-shadow layering, providing a professional "Cyber-Luxe" aesthetic


## Local Installation

1. **Clone the repository and enter into the project's directory:**
   ```bash
   git clone https://github.com/pekimnjenga/portfolio.git && cd portfolio
   ```

2. **Initialize the virtual environment**
   ```bash
   python -m venv venv
   source venv/bin/activate  # Windows: venv\Scripts\activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```
